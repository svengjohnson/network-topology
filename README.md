### Project stack

- Spring Boot 3.2.1, with Web and JPA
- H2 as database
- REST API, exposed on port 8080

### Building the project

`./gradlew build`

### Running the project

`./gradlew run`

Or after building `java -jar build/libs/ubnt-task-1.0.jar`

### Running tests

`./gradlew test`
